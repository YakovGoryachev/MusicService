Создание альбома

Альбом может принадлежать либо артисту, либо группе. Укажите только один из вариантов.

Что происходит при создании
  1. Создается запись альбома
  2. Связывается с артистом или группой
  3. Добавляются выбранные жанры
  4. Сохраняется обложка (если указана)
Важно: Укажите либо артиста, либо группу. Оба поля не могут быть пустыми.
